# exhentai-image-cralwer

## 需要解决的问题： 没有torrent时exhentai的图片保存过于繁琐,通过输入当前网站保存系列图片
## 解决问题的方式： 通过脚本爬取页面图片并保存

## 源码启动:
npm
- ```npm start```

yarn
- ```yarn start```
## 客户端打包
npm 
- ```npm build```

yarn
- ```yarn build```

打包后目录: ```./build/win-unpacked```

tip: 打包出来的项目需要配合独立的[chromium](https://www.chromium.org/)文件夹,放入打包后页面根目录
